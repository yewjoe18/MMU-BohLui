📌 About BohLui

MMU BohLui is a student expense tracking system developed using Flask, HTML, CSS, JavaScript, and SQLite database.

The purpose of this system is to help students develop better spending habits and improve financial awareness.

✨ System Features

✔ Add Expenses

Record daily expenses quickly with categories and descriptions.

✔ Expense Records

Review all spending history in one organized table.

✔ Student Login

Personalized dashboard system for each registered student.

📖 Quick User Guide

  1. Login using your student details
  2. Add your daily expenses
  3. View and manage expense records
  4. Monitor your spending habits

💡 Financial Tips

  • Avoid unnecessary purchases
