📌 About MMU BohLui

MMU BohLui is a web-based expense tracking application developed as a Mini IT Project using Flask, HTML, CSS, JavaScript, and SQLite database.

This project aims to help users improve financial awareness and spending habits through a clean and interactive system.

✨ Key Features

✔ Add Expenses

Record expenses quickly with categories and descriptions.

✔ Expense Records

View all spending history in a clean table layout.

✔ Interactive UI

Smooth user interface with JavaScript interaction and validation.

📖 How To Use

  1. Open the Add Expense page
  2. Enter your expense information
  3. Submit the expense form
  4. Review your records in Expense Records

💡 Financial Tips

  • Track your spending every day
  • Avoid unnecessary purchases
  • Separate needs from wants
  • Set monthly savings goals
