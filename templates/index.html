💸 MMU BohLui

Smart and Simple Expense Tracker for Students

📌 About the System

MMU BohLui is a web-based expense tracking system developed to help students manage their daily spending efficiently. Users can record expenses, review spending history, and monitor their financial habits in a simple and organized way.

🚀 Get Started

Use the buttons below to begin managing your expenses. You may add a new expense record or view all saved expenses.

✨ Key Features

  • ✔ Add and store expense records
  • ✔ View all expenses in a table format
  • ✔ Calculate total spending amount
  • ✔ User-friendly interface design
  • ✔ Simple navigation system

🎯 Project Objective

The objective of this project is to develop a simple expense management application using HTML, CSS, Python Flask, JavaScript, and SQLite. This system aims to improve users’ awareness of their spending patterns and financial management.

📖 User Guide

To begin using the system:

  1. Click on “Add New Expense”
  2. Enter expense details in the form provided
  3. Submit the expense record
  4. View all records in the expense list page
